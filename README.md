# Apex OOP Bank Simulation (Python) — README

## Kya hai (One-liner)
Apex OOP Bank Simulation ek educational, object-oriented Python project hai jo banking domain ke core concepts (Accounts, Customers, Transactions, interest, overdraft, transfers) ko dikhata hai aur Supabase cloud database ke saath integrate karta hai. Project me ek FastAPI-based REST API, ek Streamlit UI, aur ek CLI/demo runner (main.py) included hai.

---

### Stack
- Language: Python (3.9+ recommended)
- Frameworks / runtime: FastAPI (API), Streamlit (UI), Uvicorn (ASGI server)
- Notable libraries: supabase, python-dotenv, pydantic, pandas, requests
